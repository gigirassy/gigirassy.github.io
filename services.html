Nune's services on blitzw.in (WIP)

As true to my nature, I'm planning to host a small variety of privacy/usage services. I'll be utilizing an old laptop (April 2nd update: Desktop now) with Arch Linux installed for it. Here's a list and some updates on my plans.

Current services...

  • Comic Book Reader: Copy of Afif Zafri's comic book reader. You can read any comic book file using it. If you don't have any files, you can still test it out with a few Transformers comics.
  • EasyMDE: Copy of EasyMDE. Very barebones with no saving at all and a lack of icons. It's best to use it to build your Rentries or markdown documents easier.

Possible future services, in order to highest to lowest priority

  • ForgeJo (closed instance)
  • SimplyTranslate
  • Etherpad (replacing EasyMDE)

Privacy Policy

I am tired of privacy policies that are worded so that they enable certain corps to take your right to privacy as you either 1) don't want to read them or 2) they use so much academic language that you don't understand. I'll be straight to the point here.

Data:

  • None. My site doesn't keep any data from people viewing, including those under the age of 13.
  • In fact, this main site is a collection of static pages made of HTML and CSS with absolutely no Javascript. I don't even use a framework, and purposefully built my site to load and run properly on even the oldest browsers. I'll make a page about this sometime.
  • The comic book reader located on random.blitzw.in uses Javascript, but not any type able to track your activity. All comics read are stored locally in blobs.

Hosting:

  • I bought my domain from Gandi.net, and I configured GitHub Pages to this site, which allows this site to have the content you love and hate.
  • The domain suffix is the top-level domain for India, and India has licensed it to domain providers so sites all over the world, like mine, can use it.
  • The random.blitzw.in comic book reader uses Codeberg Pages instead of Github Pages.

External sites:

  • I'm not responsible for the data that might be stolen by external sites I might link, such as Redbubble, a pretty gosh darn crappy corp by itself. I practice basic privacy while using the web myself, and you should too.
